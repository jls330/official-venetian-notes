# official-venetian-notes

本仓库用于归档和发布多个独立的 HTML 页面。这些页面内容相互独立，不针对任何特定的域名或网站。

## 项目简介

official-venetian-notes 是一个静态 HTML 页面集合仓库。每个页面均以独立文件形式存放，可直接通过浏览器打开查看。仓库的定位是资料归档与展示，不包含动态后台或数据库依赖。

## 目录结构说明

```
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放主要 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 公共资源（样式、图片、脚本等）
    ├── css/
    ├── js/
    └── images/
```

- `index.html`：可选的仓库首页，用于导航或索引。
- `pages/`：核心 HTML 页面存放目录，每个文件为一个独立页面。
- `assets/`：页面共用的静态资源文件，按类型分目录存放。

## 页面归档说明

- 所有页面均为纯 HTML 文件，不依赖外部服务或 API。
- 页面之间没有强关联，可单独浏览或归档。
- 文件名遵循 `page-三位数字.html` 格式，便于排序和引用。
- 如需新增页面，请按照命名规范放入 `pages/` 目录，并确保所有资源引用路径正确。

## 维护说明

- 本仓库由维护者定期更新，新增或调整页面内容。
- 提交信息请尽量清晰描述变更内容（如“新增页面 page-003.html”）。
- 不建议对已有页面进行结构性修改，以免影响外部引用。
- 如有资源更新（如样式或脚本），请同时更新所有相关页面的引用路径。

## 使用方式

1. 克隆或下载本仓库。
2. 直接在浏览器中打开 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可将仓库部署到任意静态托管服务，通过 URL 访问各页面。
